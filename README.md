## Running locally

1. Clone this repo:

```sh
$ git clone https://github.com/bzenky/bzenky.dev.git
```

2. Then go to the project's folder:

```sh
cd bzenky.dev
```

3. Install all dependencies:

```sh
yarn install
```

4. Run locally:

```sh
yarn dev
```
